# Topic 1 Hackathon Delivery

## Goal

Deliver README Topic 1 as a balanced hackathon project that combines frontend beautification, backend SQL-to-Java migration, and an AI-assisted workflow showcase.

## Requirements

- Improve the visual quality and usability of the key frontend flows.
- Migrate representative report business logic out of SQL and into Java with better separation of responsibilities.
- Produce artifacts that show how AI tools and Trellis-style workflow supported the work.
- Preserve the maker/checker approval flow and export behavior.

## Acceptance Criteria

- [ ] At least 2-3 key frontend screens are visibly improved and demo-ready.
- [ ] At least 1 representative report is migrated from SQL-heavy logic to a clearer Java-based implementation.
- [ ] AI workflow artifacts exist for planning, implementation, and showcase.
- [ ] The final maker → checker → export flow can be demonstrated end-to-end.

## Technical Notes

- This is a hackathon-oriented umbrella task.
- Work is split into four child tasks: frontend beautification, backend SQL migration, AI workflow showcase, and final demo packaging.
- Prefer depth and clarity over migrating every report or redesigning every screen.
